## Overview

AI Travel Planner was my first complete Vibe Coding project. Users input their destination, budget, interests, and time constraints, and AI generates a detailed personalized itinerary.

## Core Features

- **Smart Itinerary Generation**: AI generates daily itineraries based on user preferences
- **Budget Management**: Automatic cost estimation for each activity
- **POI Recommendations**: Tag-based recommendations for attractions and restaurants
- **Itinerary Editing**: Drag-and-drop adjustment and manual modification
- **Sharing**: Generate itinerary links to share with travel companions

## Build Process

This project was built entirely through Cursor + Claude, taking approximately 12 hours from idea to working MVP. I documented the entire process in the blog post ["How I Built an AI Travel Planner in One Weekend with Cursor"](/en/blog/vibe-coding-intro).
